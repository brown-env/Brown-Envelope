# Brown Envelope - Music Project Website

A professional, responsive website for the Brown Envelope music project.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Smooth Navigation**: Single-page application with smooth scrolling
- **Music Showcase**: Display your latest releases and discography
- **Contact Form**: Interactive form for fans and booking inquiries
- **Social Media Integration**: Links to all your social media platforms

## Structure

- `index.html` - Main website file with all sections
- `styles.css` - Complete styling and responsive design
- `script.js` - Interactive features and smooth scrolling

## Sections

1. **Home/Hero**: Eye-catching introduction with call-to-action buttons
2. **About**: Tell your story and connect with your audience
3. **Music**: Showcase your releases with links to streaming platforms
4. **Contact**: Contact form and social media links

## Deployment

This website is ready to deploy to any web hosting service:

### Option 1: GitHub Pages
1. Go to your repository settings
2. Navigate to "Pages" section
3. Select the branch to deploy (e.g., main or copilot/set-up-music-project-website)
4. Your site will be available at `https://yourusername.github.io/Brown-Envelope`

### Option 2: Netlify
1. Connect your GitHub repository to Netlify
2. Deploy the site (no build steps required)
3. Configure your custom domain

### Option 3: Any Web Host
Simply upload the files (index.html, styles.css, script.js) to your web hosting service.

## Customization

### Update Content
- Edit `index.html` to update text, links, and structure
- Update the hero title, about section, and music links
- Add your actual social media links in the contact section

### Update Colors
- Modify the CSS variables in `styles.css`:
  - `--primary-color`: Main brand color (default: brown)
  - `--secondary-color`: Accent color
  - `--dark-color`: Dark backgrounds

### Add Real Music
- Replace placeholder artwork with actual album covers
- Update streaming platform links
- Add embedded players (Spotify, SoundCloud, etc.)
